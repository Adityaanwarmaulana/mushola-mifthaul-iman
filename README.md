# Website Mushola Miftahul Iman

Website resmi Mushola Miftahul Iman  
Dibangun menggunakan Laravel.

## Fitur
- Website publik
- Login admin
- Manajemen berita
- Jadwal sholat
- Halaman donasi

## Teknologi
- Laravel
- Tailwind CSS
- MySQL

## Cara Install (Local)
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` ke `.env`
4. Jalankan `php artisan key:generate`
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
